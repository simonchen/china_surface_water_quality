# china_surface_water_quality
中国地表水质实时监测

# 使用方法
安装python 3.0 以上版本，并且安装flask模块

**Linux**
```
pip install flask
```
**Windows**
```
python -m pip install flask
```

**启动**
```
python cnemc.py
```

**打开浏览器, 输入 http://127.0.0.1:9898 查看**
